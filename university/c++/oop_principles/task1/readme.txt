1. Open the terminal and locate to the directory containing this text file

2. Run the command: g++ -std=c++11 -o launcher *.cpp && ./launcher

3. All the .txt / .4 files are stored in the 'save' folder

4. The commands supported are:
	
	- 'save FILENAME'
	
	- 'load FILENAME.EXTENSION'
	
	- 'display_recomposition'
	
	- 'display_tree'
	
	- 'quit'
